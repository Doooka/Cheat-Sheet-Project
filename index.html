Cheat Sheet Project: Basic CSS Selectors

Selector Example Example Description
element p { } Selects all <p> elements
universal * {font-family: Verdana;} Selects all elements
class .brand { } Selects all elements with class="brand"
id #brand { } Selects the element with id="brand"
attributes [href] { } Selects all elements with an href attribute
pseudo-classes p:hover { } Selects all elements with pseudo-class of "hover"
Chaining(element.class) p.media { } Selects all <p> elements with class="media"
Descendant Combinator(class element) .media p { } Selects all <p> elements that are a descendants of elements with class="media"
Multiple Selectors(element, element) h1, p { } Selects all <h1> and <p> elements
